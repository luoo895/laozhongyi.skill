# 中医经方论治经验库

TCM Pattern Prescriptions & Treatment Experience

> 汇集常用中医方剂及其加减应用，涵盖内、外、妇、男科等多科病证的辨证论治经验。

## 📚 项目简介

本项目整理了临床常用的中医方剂，包括：

- **痛风方** - 祛风除湿，通络止痛
- **疝气方** - 温肝散寒，行气散结
- **面瘫方** - 益气活血，祛风化痰
- **纤维瘤方** - 疏肝理气，软坚散结
- **糖尿病方** - 益气养阴，清热降糖
- **高血压方** - 平肝潜阳，化痰通络
- **更年期综合征方** - 疏肝清热，滋阴降火

## 🎯 项目目标

1. **经验传承** - 将临床常用方剂系统整理，便于学习参考
2. **辨证论治** - 总结中医辨证思路和用药规律
3. **配伍心得** - 提炼常用药物配伍规律
4. **开放共享** - 通过开源方式促进中医知识传播

### 配伍规律

#### 1. 祛风湿痹
- 黄芪 + 防己 + 白术（防己黄芪汤）
- 苍术 + 黄柏 + 薏苡仁（二妙丸 + 四妙丸）
- 独活 + 桑寄生 + 杜仲（独活寄生汤）

#### 2. 散结消癥
- 夏枯草 + 生牡蛎 + 玄参（消瘰丸）
- 海藻 + 昆布 + 黄药子
- 山慈菇 + 蚤休

#### 3. 疏肝理气
- 柴胡 + 黄芩（小柴胡汤）
- 橘叶 + 青皮 + 陈皮
- 小茴香 + 吴茱萸 + 川楝子（暖肝煎）

#### 4. 清热利湿
- 龙胆草 + 栀子 + 黄柏（龙胆泻肝汤）
- 茵陈 + 车前子 + 泽泻
- 苦参 + 地肤子 + 白鲜皮

#### 5. 活血化瘀
- 桃仁 + 红花 + 川芎（桃红四物汤）
- 丹参 + 赤芍 + 当归
- 鬼箭羽 + 鸡血藤 + 地龙

#### 6. 益气养阴
- 黄芪 + 党参 + 白术（四君子汤）
- 生地 + 麦冬 + 玄参（增液汤）
- 枸杞子 + 女贞子 + 旱莲草（二至丸）
